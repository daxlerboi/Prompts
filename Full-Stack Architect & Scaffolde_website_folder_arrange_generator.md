# 🧠 Universal Software Architecture Prompt

## **Prompt**

Act as a **Senior Software Architect**.

I want to build a **[Insert Project Name, e.g., 2D Platformer Game or ML Image Classifier]**.

---

## **Instructions**

Please provide the following **in order**:

### **1. Complete Folder Structure**
- Show a clean, scalable, production-ready directory layout.

---

### **2. Core Files with Code**
- Provide **individual code blocks for each core file**
- Include:
  - File name
  - Full code
  - Clear explanation of the logic

---

### **3. Project Scaffolding Script (Python)**
- Provide a **single Python script** that:
  - Recreates the entire folder structure
  - Injects all provided code into the correct files

- Set the root directory to:  
  **[Insert Path, e.g., `D:\MyNewProject`]**

---

### **4. Dependency Management**
- Include:
  - `requirements.txt` (for Python projects)  
  **OR**
  - `package.json` (for JavaScript projects)

---

### **5. Code Quality Requirements**
Ensure all code is:
- Production-ready  
- Modular  
- Clean and well-structured  
- Properly commented  

---

## **Optional Pro Addition (Recommended)**

Also include:
- Terminal commands to:
  - Initialize a virtual environment
  - Install all dependencies required for this project
